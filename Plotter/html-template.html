[Function]
fplot pane domain fn &rest args => result


Functional plotting in pane of the function over the indicated domain. - Example: ;;; (plt:fplot 'plt '(-20 20) (lambda (x) (/ (sin x) x)) ;;; :clear t :thick 2 :title "Sinc(x)") where 'plt is the name of the desired plotting window. It will be created if not already visible on screen.


[Function]
histogram pane v &rest args => result



[Function]
paramplot pane domain xfn yfn &rest args => result


Parametric ploting in pane of the xfn and yfn functions over the indicated domain. All functional plotting is really performed as parametric plotting. Parametric plotting allows for non-single-valued functions. - Example: ;;; (plt:paramplot 'plt '(0 1) ;;; (lambda (phi) (cos (* 2 pi phi))) ;; the X-function ;;; (lambda (phi) (sin (* 2 pi phi))) ;; the Y-function ;;; :clear t :thick 2 :title "Circle") where 'plt is the name of the desired plotting window. It will be created if not already visible on screen.

[Function]
plot-image pane xv yv image-arr &rest args => result


Plot an image in the specified pane using the sequences xv and yv for the X and Y scales. The image array holds the z-values.


[Function]
read-image &optional filename => result


Read an external image file. If the filename argument is elided, CAPI will prompt for one.


[Function]
render-image pane ext-img &rest args => result


Render an external image in the specified window pane.

[Function]
tvscl pane arr &rest args => result


Display an image array of z-values in the specified pane. This is distinct from plotting an image with user specified X and Y values. TVSCL always uses implicit pixel coordinates for its X and Y values.
